## Sobre o projeto

Este projeto foi desenvolvido com o objetivo de criar uma presença profissional na web, reunindo informações sobre habilidades, tecnologias e projetos em uma interface moderna e organizada.

O portfólio funciona como uma vitrine digital, permitindo apresentar conhecimentos em desenvolvimento web e compartilhar trabalhos realizados durante os estudos.

## Tecnologias utilizadas

* HTML5

* Git
* GitHub
* GitHub Pages

## Funcionalidades

* Apresentação profissional

## Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/daniellydev/portfolio_dev.git
```

2. Acesse a pasta do projeto:

```bash
cd portfolio_dev
```

3. Abra o arquivo `index.html` em seu navegador.

## Objetivo

Desenvolver um portfólio pessoal para consolidar conhecimentos em HTML e CSS, além de criar um espaço para divulgar projetos e acompanhar a evolução como desenvolvedora Front-end.
